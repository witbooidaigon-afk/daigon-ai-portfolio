# 1 · Executive Summary

**Daigon (operating as Dynamic Media)** designs, builds, deploys, and maintains **AI-powered automation systems** for service businesses. We do not simply "write code" — we install revenue-protecting infrastructure: systems that answer the phone, follow up on leads in seconds, recover missed calls, book appointments, and run the repetitive back-office work that owners and staff no longer should.

Our delivery engine is a production **n8n automation platform** (self-hosted on a dedicated VPS) orchestrating best-in-class AI models and communication channels into reliable, always-on business workflows.

**The problem we solve:** Most service businesses lose money daily — not from bad marketing, but from slow, manual, broken follow-up. Leads wait hours for a reply and buy from whoever responds first. Missed calls are never returned. Owners burn hours on booking, reminders, and quotes. We close those gaps with automation that runs 24/7, responds in seconds, and never forgets a follow-up.

# 11 · Architecture Gallery

**Platform overview**
```
 Triggers ──►  n8n (self-hosted VPS): webhooks · schedules · IF/switch · code · retries  ──► Channels
 • Inbound/missed call          ┌── AI LLMs (GPT / Claude) ──┐                    • SMS (Twilio)
 • Form / new Sheet row         │   Voice (Retell/Vapi+11L)  │                    • WhatsApp (Meta)
 • WhatsApp / Cal.com / webhook └── ASR (Whisper) ───────────┘                    • Email · Slack · Voice
                          Data: Sheets · Airtable · Notion · Postgres   Calendar: Cal.com
```

**AI Voice Agent**
```
Caller ─► Retell/Vapi (Whisper ASR) ─► LLM (GPT-4o + KB) ─► Book (Cal.com) / FAQ / Escalate(Slack)
       ─► n8n webhook ─► CRM record ─► team notification + call summary
```

**Missed-Call & No-Show Recovery**
```
Missed call ─► n8n ─► Twilio SMS (seconds) + Slack alert ─► no-reply ─► follow-up sequence
No-show (calendar) ─► recovery sequence ─► rebook link (Cal.com)     [live screenshots exist]
```

**Speed-to-Lead**
```
New lead ─► n8n ─► enrich + qualify (LLM) ─► instant SMS+WhatsApp+Email ─► hot? route+book ─► CRM write
```
